WowMenu

Discover menus, explore flavors with WowMenu

Screenshots

WowMenu is a mobile application that allows users to find nearby restaurants and browse their menus and menu boards. Restaurant owners can also create digital menu cards that are visible to app users. The app shows all restaurants within a 5km radius.

Features

  • User authentication using Google.
  • Nearby restaurant search with location tracking.
  • Browse restaurant menus and menu boards.
  • Filter menu items based on dietary preferences.
  • Restaurant owners can create and manage digital menu cards.
  • Dark mode support.
  • User-friendly interface.
More About Us at https://wowmenu.github.io/additional-docs/
